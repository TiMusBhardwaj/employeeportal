

# Project Title

Employee management system using Spring State machine.


## BUILD Using Maven

```mvn clean package```

#### Api DOC Reference is available 

http://localhost:8080/swagger-ui/index.html



## API Reference

#### Create Employee

```http
  POST /employee
```


#### Udate employee status

```http
  PUT /employee/${id}/status
```
#### GET employee 

```http
  Get /employee/${id}
```
